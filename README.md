# CS2ToGO
Bring your **Counter-Strike 2 inventory** into **legacy CS:GO servers** after official services have shut down.

This server plugin allows players to use their **existing CS2 skins and loadouts** inside CS:GO by querying their inventory, identifying skins that are **compatible with CS:GO weapons**, and letting players equip them in-game.

---

## Features

### CS2 Inventory Integration
- Queries a player's **CS2 inventory**
- Detects **CS:GO-compatible skins**
- Automatically makes those skins available for use in the server

### In-Game Skin Menu
- Players can open the skin selector with: !skins
- When purchasing a weapon in-game, players can **equip any compatible skin** from their CS2 inventory.

### Loadout Swapping
Players can customize their weapon loadout using: !loadout
Supported swaps include:

| Default Weapon | Alternative |
|----------------|-------------|
| P2000 | USP-S |
| M4A1 | M4A1-S |
| Five-Seven / TEC-9 | CZ75 |
| MP7 | MP5 |

This allows players to match their **CS2 loadout preferences** while playing legacy CS:GO.

## Goal

With official CS:GO inventory services shut down, this project preserves the ability for players to **continue using their skins and preferred loadouts** while playing on legacy CS:GO servers.
